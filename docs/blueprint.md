# **App Name**: KimiPK Redefined

## Core Features:

- Product Loading: Asynchronously load product data from a local products.json file, always sorted newest-first.
- Hero Section: Implement an interactive hero section with parallax and mouse-tracking effects powered by Swiper.js.
- Infinite Scroll: Seamlessly load and animate additional products as the user scrolls down the home and category pages, mimicking Shopify's smooth loading experience.
- Slide-Out Cart Drawer: Implement a slide-out cart drawer for managing cart items without leaving the current page.
- Intelligent Search: A full-screen search overlay with typo-tolerant results.
- WhatsApp Order Submission: Compile order details into a URL-encoded string and submit via WhatsApp.
- Related Product Recommendations: AI powered tool that generates cross-selling recommendations, 'You May Also Like' section of the product detail page.
- Theme System: Implement a light/dark theme system that defaults to the user's system preference and persists the user's choice in localStorage.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) for a sense of sophistication and modernity.
- Background color: Light gray (#F0F0F0) for a clean, minimalist feel.
- Accent color: Muted Purple (#800080) to complement the primary and add visual interest.
- Headline font: 'Inter', a sans-serif font for a modern feel.
- Body font: 'Roboto', a sans-serif font for maximum readability.
- Responsive layout with tailored experiences for mobile, tablet, and desktop.
- Scroll-triggered animations, subtle hover states, and elegant skeleton loaders for a fluid, delightful user experience. Ensure picture loading animations are smooth, like Shopify's.
